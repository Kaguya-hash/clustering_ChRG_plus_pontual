Understanding Clustering

What is Clustering?

Clustering is a core methodology in unsupervised machine learning. Its primary goal is to partition a dataset into subsets—known as clusters—so that elements within the same group are more similar to each other than to those in different groups.

Despite this simple formulation, the concept of similarity can be defined in many different ways, which leads to various families of clustering algorithms.

Common Clustering Approaches

Different algorithms capture different types of relationships in the data. Here are some example clustering methods:

  • K-means: A partitional method that uses centroids.
  • DBSCAN: A density-based method that finds clusters of any shape.
  • Hierarchical Clustering: A method that builds a tree of clusters.
  • Mean Shift: A centroid-based algorithm that shifts points toward the mode.
  • Affinity Propagation: A method based on "message passing" between data points.

Many of these methods and more can be found here: scikit-learn clustering documentation. This is the official clustering module for scikit-learn, which is a popular machine learning library for Python.

Evaluating Clusters: The Adjusted Rand Index (ARI)

How do we know if our clustering algorithm did a good job? One of the most robust metrics is the Adjusted Rand Index (ARI). The ARI quantifies the agreement between two partitions—typically, the partition generated by our algorithm and a known true reference partition. It goes a step further than basic accuracy by correcting for chance.

A score of 1.0 means there is a perfect match between the algorithmic clusters and the true clusters. A score near 0.0 means the clustering is no better than random assignments.

Note: When dealing with algorithms that detect noise, points identified as outliers are usually excluded from the ARI calculation, as they do not formally belong to any structural cluster. Because of this, ARI should often be analyzed alongside a Coverage metric, which measures the proportion of data successfully clustered versus discarded as noise.
